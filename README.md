# Hangman-on-calculator
A simple and fun Hangman game designed to run on a Casio Graph Math+ calculator.

## Description
This project is a functionnal, adapted to be played on a Casio Graph Math+ calculator. The goal is to find a word defined bythe user.

## Features
- **Define a word:** At the start of the game you have to choose a word with a max of 63 letters.
- **Find the world:** Play with your friends and find the world they hid.
- **Win or Lose:** You can do 10 mistakes max. But if you find the world before, you win.

## Requirements
A **Casio Graph Math+ calculator.**

Required libraries to run Python scripts on the calculator: **casioplot**, **turtle** and **random**.

## Installation
1. Download the files from this repository.
2. Transfer them to your calculator.
3. Run the Hangman.py file to start the game.

## How to Play
- Start the Game: Enter a word to begin.
- Enter letters: Press `keys` to complete the word.
- Finish the game : Complete the word or do 10 mistakes to win or lose.
